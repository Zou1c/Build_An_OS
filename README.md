### 实验启动方法
找一个某一天的小项目(e.g. day5文件夹下的harib002)

执行!cons_nt.bat会执行cmd命令，于是就将shell的工作目录引导到当前小项目下了(当然你也可以手动cd 到这里)  

然后运行  
```
make run
```
就可以在QEMU中“看”到这次实验的效果了
